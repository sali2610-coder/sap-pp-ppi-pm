# SAP SD — Order-to-Cash quiz

**1. Which tcode creates a sales order?**  _(remember)_

   A. VA01
   B. VA03
   C. VL01N
   D. VF01

**2. Which table stores outbound delivery headers?**  _(understand)_

   A. LIKP
   B. VBAK
   C. VBRK
   D. BKPF

**3. Billing (VF01) creates an FI document.**  _(understand)_

   A. True
   B. False

**4. List the three core O2C transactions in order.**  _(apply)_


---

## Answer Key

1. **A** — VA01 = create; VA03 = display; VL01N = delivery; VF01 = billing. [Ch.1]
2. **A** — LIKP = delivery header; VBAK = order header; VBRK = billing header; BKPF = FI header. [Ch.2]
3. **True** — Billing passes revenue to FI, creating BKPF/BSEG entries. [Ch.3]
4. **VA01 (order) → VL01N (delivery) → VF01 (billing).** [Ch.1-3]
