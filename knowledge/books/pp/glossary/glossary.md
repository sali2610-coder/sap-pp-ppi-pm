# PP Glossary — bilingual (SAP terms kept English)

| Term (EN, verbatim) | הסבר בעברית |
|---|---|
| `MRP Live (MD01N)` | הרצת תכנון דרישות חומר על מנוע HANA — מהירה, מתעדפת חומרים שהשתנו. |
| `PLKO` / `PLPO` | כותרת ופעולות של רשימת משימות / מתכון ראשי (Routing / Master Recipe). |
| `MAPL` | שיוך רשימת משימות/מתכון לחומר (Material–Task List Assignment). |
| `MKAL` | גרסת ייצור (Production Version) — חובה ב-S/4HANA. |
| `AFKO` / `AFPO` / `AFRU` | כותרת פק"ע / פריט פק"ע / רשומות דיווח ביצוע (Confirmation). |
| `Control Recipe` / `PI Sheet` | מתכון בקרה וגיליון הוראות תהליך בייצור תהליכי (PP-PI). |
| `LOIPRO` | IDoc להפצת פק"ע ייצור למערכות ביצוע (Zetes/Daymax ב-CBC). |
| `Backflush (MFBF)` | ניכוי רכיבים אוטומטי בעת דיווח תוצר (ייצור חוזר). |
| `Kanban Control Cycle (PK01)` | מעגל בקרה לחידוש מונחה-משיכה (Pull). |
| `DDMRP` | Demand-Driven MRP — תכנון מבוסס באפרים וצריכה בפועל. |
| `pMRP` | Predictive MRP — סימולציית קיבולת/ביקוש לפני תכנון תפעולי. |
| `Batch Determination (COB1)` | קביעת אצווה אוטומטית באמצעות Condition Technique. |
| `CO01 / CO02 / CO11N` | יצירה / שחרור / דיווח ביצוע של פק"ע ייצור בדיד. |
| `COR1 / CORK` | יצירה / דיווח-סגירה של פק"ע ייצור תהליכי. |
| `MD61` | דרישות עצמאיות מתוכננות (Planned Independent Requirements / PIR). |

> Source: *Production Planning with SAP S/4HANA*. Transformative bilingual glossary — identifiers verbatim, explanations in Hebrew.
