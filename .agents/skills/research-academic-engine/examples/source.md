# Long-Run Returns to Early Childhood Investment (working notes)

## 1. Context
Early childhood interventions show strong short-run cognitive gains. Whether these persist into
adulthood, and through which channel, is debated.

## 2. Theory
Cunha and Heckman (2007) argue skills are self-productive and complementary over time — "skills beget
skills" — so early investment raises the return to later investment.

## 3. Evidence
Heckman et al. (2013) report that the HighScope Perry Preschool Program raised adult earnings and
lowered criminal involvement, with returns concentrated among the most disadvantaged.

## 4. Gap
Few studies link field-level program fidelity to long-run outcomes; mechanisms remain contested.
