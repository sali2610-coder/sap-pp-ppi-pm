# Glossary — SAP SD Order-to-Cash

Definitions drawn only from the source. Codes, tables, and movement types are preserved verbatim.

- **BKPF / BSEG** — Tables holding the FI (Financial Accounting) document created when billing passes revenue to Finance. `BKPF` is the accounting header, `BSEG` the accounting line items.
- **Billing** — Step that creates the invoice sent to the customer and passes the revenue to Financial Accounting. Create: `VF01`; header table `VBRK`, items `VBRP`.
- **FI document** — The Financial Accounting document generated by billing when revenue is passed to Finance. Stored in `BKPF`/`BSEG`.
- **Goods issue** — The posting that records a material movement (movement type `601`) and reduces inventory; triggered by the outbound delivery.
- **LIKP / LIPS** — Tables for the outbound delivery: `LIKP` is the header, `LIPS` the items.
- **Movement type 601** — The material movement type posted by goods issue; it reduces inventory.
- **Order-to-Cash** — The sales flow from sales order to billing: sales order (`VA01`) → outbound delivery (`VL01N`) → billing (`VF01`).
- **Outbound delivery** — The logistics document that triggers picking and goods issue. Create: `VL01N`; header table `LIKP`, items `LIPS`.
- **Picking** — Activity triggered by the outbound delivery to prepare goods for shipment (precedes goods issue).
- **Sales order** — A customer request to deliver goods or services. Create: `VA01`, change: `VA02`, display: `VA03`; header table `VBAK`, items `VBAP`. References the customer (sold-to party) and one or more materials with quantities and prices.
- **Sold-to party** — The customer that the sales order references.
- **VA01 / VA02 / VA03** — Transaction codes to create / change / display a sales order.
- **VBAK / VBAP** — Tables for the sales order: `VBAK` is the header, `VBAP` the items.
- **VBRK / VBRP** — Tables for the billing document: `VBRK` is the header, `VBRP` the items.
- **VF01** — Transaction code to create the billing document (invoice).
- **VL01N** — Transaction code to create the outbound delivery.
