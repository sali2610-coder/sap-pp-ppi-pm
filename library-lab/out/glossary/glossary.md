# SAP Core Modules Library — Glossary

_10 terms._

- **BKPF** — Accounting document header table (FI).
- **BSEG** — Accounting document line-item table (FI).
- **EKKO** — Purchasing document header table.
- **ME21N** — Transaction to create a purchase order (MM).
- **MIGO** — Transaction to post a goods movement (e.g. goods receipt).
- **MIRO** — Transaction for invoice verification (MM).
- **VA01** — Transaction to create a sales order (SD).
- **VBAK** — Sales order header table.
- **VF01** — Transaction to create a billing document (invoice).
- **VL01N** — Transaction to create an outbound delivery.
